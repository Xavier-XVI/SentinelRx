# SentinelRx — Patient Profile Template

> Instructions: Fill in every field. Save this file as `patients/[firstname-lastname].md`.
> Update the LAST_UPDATED date after every prescription change, physician visit, or hospitalization.
> One patient file in the `patients/` folder at a time.

---

## Profile Metadata

```
LAST_UPDATED: YYYY-MM-DD
STALENESS_THRESHOLD_DAYS: 14
```

> `STALENESS_THRESHOLD_DAYS`: SentinelRx flags stale profiles after this many days. Default is 14.
> For patients with frequent prescription changes (e.g., warfarin INR monitoring, post-hospitalization),
> consider reducing to 7. Update this value to match your patient's real-world prescription change frequency.
